# Liquid Scroll Gallery  
スクロールに合わせて画像が“液体のように歪む”ミニギャラリー。  
1枚の画像だけで成立する、軽量でモダンな Web 表現。

---

## 📸 プロジェクト概要
このミニプロジェクトは、**スクロールに応じて画像が波のように変形するエフェクト**を実装した、1ページ完結のギャラリーです。

- SVG フィルタ（feTurbulence / feDisplacementMap）を使用  
- スクロール割合から波の強さを計算  
- 画像1枚で成立するシンプル構成  
- マウス操作なしで自然に動く “液体アニメーション”

---

## 📂 ファイル構成
index.html … HTML/CSS/JS が1つにまとまったシンプル構成

yaml
コードをコピーする

## 🛠 必要なもの
- 特になし（ブラウザだけでOK）
- ライブラリ不要
- 画像を差し替える場合は `<img src="…">` を変更するだけ

---

## 🚀 使い方
1. `index.html` をブラウザで開く  
2. ページを上下にスクロール  
3. 画像が “ぷにゃ〜っ” と歪んで戻る動きを楽しむ  

---

## 🔧 カスタマイズ項目

### 波の強さ
js
const wavePower = Math.sin(scrollRate * Math.PI) * 50;
50 → 最大変位

数値を下げるとおとなしく、上げると激しくなる

## 揺れの速さ
js
コードをコピーする
const time = scrollY * 0.002;
0.002 を大きくすると揺れが速くなる
小さくすると静かめに

## 背景色
css
コードをコピーする
body {
  background: #111;
}
白など好みに変更可能。

##画像の差し替え
html
コードをコピーする
<img src="your-image.jpg">
